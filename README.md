## 아몰랑파킹 (AmorlangParking)

<br>

## 개발 배경
아파트 주차문제에 대한 갈등이 심해지면서 이에 대한 문제점 해소가 필요하다고 판단

<br>

## 팀원

* Front-End : 김용성, 이진희
* Back-End : 임규원, 정겨운

<br>

## 사용 스택
Spring Boot, Spring Security, JPA
